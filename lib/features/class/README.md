# features/class

## 概要

- クラス（学級）の作成・編集・削除
- クラス詳細・メンバー管理（追加・削除）
- 担任選択（作成者がデフォルト）
- 園クラス（kindergartenId）との紐付け
- Firestore連携

## 主なファイル

- `view/` ... クラス一覧・詳細・作成・編集画面
- `view/widgets/` ... メンバーリスト・追加ダイアログ等
- `view_model/` ... クラス状態管理（Riverpod）
- `repository/` ... Firestore連携
- `model/` ... クラスモデル

## 備考

- クラス編集・作成時に担任・園クラスを選択可能
- メンバー追加は園児マスタ・保護者検索ダイアログ
